Ejemplo 1 - Petición asíncrona, método Get y retorno de subvista: En este ejemplo, al hacer click, se realiza una petición Get al servidor que retorna una vista (ejemplo1.html) que será cargada en una capa (div) utilizando la función $.load de JQuery. La vista ejemplo1.html se considera en estos casos una subvista. Ver Ejemplo

Ejemplo 2 - Petición asíncrona, método Post y retorno de subvista: En este ejemplo, se envían los datos de forma asíncrona por AJAX y la vista de respuesta es cargada en una capa div.


Ejemplo 3 - Petición asíncrona, método Post y retorno de JSON: En este ejemplo, se envían los datos de forma asíncrona por AJAX y a diferencia de los ejemplos anteriores, la respuesta es un JSON con el cual desde JavaScript se toma decisiones modificando el DOM. Este tipo de uso de AJAX es de los más frecuentes, ya que con los datos en un JSON se puede aplicar lógica de vista modificando cualquier región del DOM y siendo este, un mecanismo más optimo y ligero. En código de servidor se forma una String con formáto JSON.


Ejemplo 4 - Petición asíncrona, método Get y retorno de JSON: La diferencia con el ejemplo anterior, más allá de que se explota el método Get, es que la respuesta JSON se genera serializando un objeto directamente a código JSON.


Ejemplo 5 - Petición asíncrona, método Post y retorno de JSON: En este ejemplo, permitimos abrir una ventana modal de Jquery UI y se muestra un ejemplo de como podemos retroalimentar al usuario mientras se ejecuta una acción mostrando una animación de espera.
